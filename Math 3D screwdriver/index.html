1. Introduction

In this project, I was asked to use online graphing tools to visualize quadric surfaces in 3D space, and eventually I was expected to use these surfaces to design an object in the real world.

The goal of this project was to help me:

  • Tbe familiar with using online graphing tools to visualize 3D shapes.
  • connect math equations with 3D shapes in the real life.
  • organize your thoughts in writing.

5. Process and Methodology

  • Read and learn about quadric surfaces from textbook and online resources.
  • Learn how to plot surfaces using CalcPlot3D from online resources
  • Pick an object of I my choice and approximate its shape with 3d surfaces
  • Design my quadratic equations and plot the object.
